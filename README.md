# Fabex Schneider Synth

A browser-based synthesizer built with HTML, CSS, and the Web Audio API.

## Features

- Polyphonic keyboard with mouse and computer-key support
- Presets for pads, leads, bass, and keys
- Waveform switching
- ADSR envelope controls
- Filter cutoff, resonance, and filter envelope
- LFO modulation
- Delay and reverb
- Drive, unison, detune, octave shift, and glide
- Live waveform visualizer

## Run locally

Open `index.html` in a modern browser and click `Start Audio`.

## Deploy with GitHub Pages

Upload these files to a GitHub repository and enable GitHub Pages from the repository settings.
